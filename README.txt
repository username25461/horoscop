Google Tag & FB pixel:

На лендинге уже есть установленный код Google Tag, FB pixel, TikTok Pixel.

На action кнопке Звонка установлены функции отстука конверсии
- fbq('track', 'Lead');
- gtag_report_conversion();

Чтобы использовать свой FB_PIXEL_ID в коде пикселя, нужно добавить
к ссылке на ваш сайт Query параметр sub5
например:
https://your-site.com/?sub5=123
где 123 - ID пикселя

Чтобы использовать свой ID в коде Google tag нужно добавить к ссылке на ваш сайт Query параметр sub4
например:
https://your-site.com/?sub4=AW-123456789/abcdfg
где AW-123456789/abcdfg - ID Google Tag
При инициализации кода используется первая часть до /
При конверсии - всё значение без разделения.

Чтобы использовать свой TikTok Pixel ID в коде пикселя, нужно добавить
к ссылке на ваш сайт Query параметр sub3
например:
https://your-site.com/?sub3=123
где 123 - ID пикселя

Для того, чтобы использовать все метрики нужно добавить все параметры в Query
например
https://your-site.com/?sub3=123&sub4=AW-123456789/abcdfg&sub5=123
